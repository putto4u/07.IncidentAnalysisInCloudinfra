# 100.4 최신 공격 기법과 포렌식의 한계 극복

## 1. 안티 포렌식(Anti-Forensics)의 진화
공격자들은 분석가의 추적을 따돌리기 위해 고도화된 안티 포렌식 기법을 사용합니다.
- **Timestomping**: 파일의 생성/수정 시간을 조작하여 타임라인 분석을 교란합니다.
- **Wiping**: 데이터를 복구 불가능하게 영구 삭제합니다.
- **Steganography**: 이미지나 오디오 파일 내부에 데이터를 숨깁니다.
- **Artifact Cleaning**: 이벤트 로그, 프리페치 등 주요 흔적을 자동 삭제하는 스크립트를 실행합니다.

## 2. 파일리스(Fileless) 공격과 메모리 포렌식
전통적인 디스크 기반 분석만으로는 탐지할 수 없는 공격이 증가하고 있습니다.
- **공격 방식**: PowerShell, WMI, 레지스트리 등을 활용하여 악성 코드를 메모리 상에만 상주시켜 실행합니다.
- **대응 전략**: 'Live Forensics'가 필수적입니다. 전원을 끄지 않은 상태에서 메모리 덤프를 수행하여 실행 중인 프로세스, 네트워크 연결, 주입된 코드(Code Injection)를 분석해야 합니다.

## 3. 암호화 및 클라우드 환경의 도전
- **Full Disk Encryption (FDE)**: BitLocker, FileVault 등으로 암호화된 디스크는 복호화 키 없이는 분석이 불가능합니다. 라이브 상태에서 키를 추출하거나 사용자 협조를 구해야 합니다.
- **Cloud Forensics**: 물리적 접근이 불가능한 클라우드 환경에서는 CSP(Cloud Service Provider)가 제공하는 로그(CloudTrail, VPC Flow Logs 등)와 스냅샷 기능을 활용한 새로운 분석 방법론이 필요합니다.

## 4. 전문가를 위한 분석 고도화 방안
- **상관관계 분석 (Correlation)**: 하나의 증거에 의존하지 않고, 여러 소스(네트워크 로그, OS 아티팩트, EDR 데이터)를 교차 검증하여 모순점을 찾아냅니다.
- **저수준 분석 (Low-level Analysis)**: 파일 시스템의 메타데이터(MFT, Journal)를 직접 분석하여 삭제된 흔적이나 조작된 데이터를 복구합니다.
- **위협 인텔리전스(CTI) 활용**: 최신 공격자의 TTPs를 미리 파악하여 분석의 우선순위를 정합니다.
