# 100.6 기업 내 포렌식 역량 내재화 전략

## 1. 내재화의 필요성
외부 전문가에게만 의존할 경우 초기 대응 속도가 늦어지고 비용이 과다하게 발생합니다. 상시적인 내부 포렌식 역량은 보안 성숙도의 지표입니다.

## 2. 포렌식 도구(S/W, H/W) 선정 가이드
- **소프트웨어**:
    - **Commercial Tools**: EnCase, FTK, Magnet AXIOM (강력한 기능, 법적 신뢰성 높음)
    - **Open Source Tools**: Autopsy, Volatility, Wireshark (비용 절감, 특정 기능 특화)
- **하드웨어**:
    - **Write Blockers**: Tableau, WiebeTech (원본 훼손 방지 필수 장비)
    - **Duplicators**: 고속 이미징 장비
    - **Workstations**: 고성능 CPU, 대용량 RAM 및 NVMe 저장 장치를 갖춘 전용 분석 PC

## 3. 전문 분석 인력 양성 로드맵
- **기초 교육**: OS 커널 이해, 파일 시스템 구조, 네트워크 프로토콜.
- **심화 교육**: 리버싱(Reverse Engineering), 메모리 포렌식, 모바일 포렌식.
- **자격증 취득**: EnCE, ACE, GCFE/GCFA, 국과수 디지털포렌식전문가 등.

## 4. 사고 대응 거버넌스 수립
- **R&R 정의**: 보안팀, 법무팀, IT 운영팀, 홍보팀 간의 협력 체계 구축.
- **Forensic Readiness**: 평소에 증거 수집이 용이하도록 로깅 정책을 표준화하고 점검합니다.
- **Playbook 제작**: 시나리오별(랜섬웨어, 내부 유출 등) 포렌식 절차를 매뉴얼화합니다.

## 5. 지속적인 환경 개선
- **Sandbox 구축**: 악성코드 분석을 위한 격리된 네트워크 환경 마련.
- **Knowledge Base**: 과거 분석 사례와 최신 위협 정보를 아카이빙하여 지식을 공유합니다.

기업 포렌식은 단순한 '수사'가 아니라 '비즈니스 연속성(BCP)'을 보장하기 위한 전략적 투자입니다.
