# 난독화 해제(De-obfuscation) 실습
## 1. 개요
난독화된 파워쉘 코드를 분석 가능한 형태로 되돌리는 실제 과정을 학습합니다. 코드의 실제 기능을 파악하기 위한 필수 단계입니다.

## 2. 상세 분석 내용
### `Invoke-Expression(IEX)` 추적 및 실제 실행 코드 추출
- **IEX 분석:** 난독화된 코드가 최종적으로 실행될 때 호출하는 `IEX`를 가로챕니다.
- **코드 추출 기법:** `IEX` 대신 `Write-Host`나 `Out-File`로 명령어를 변경하여 실행하면 실제 실행될 평문 코드가 출력됩니다.
### 파워쉘 디버깅 및 로깅 기능을 이용한 코드 복원
- **디버깅 실습:** PowerShell ISE나 VS Code 디버거를 사용하여 코드를 한 줄씩 실행하며 변수에 할당되는 복호화된 값을 확인합니다.
- **이벤트 로그 활용:** 윈도우 이벤트 로그(4104)에 기록된 실행 직전의 스크립트 블록 내용을 확인하여 복원합니다.

## 3. 대응 및 방어 전략
- **분석 자동화:** CyberChef나 전문 파워쉘 디코딩 스크립트를 활용하여 반복되는 해제 과정을 자동화합니다.
- **샌드박스 활용:** 실제 실행이 수반될 수 있으므로 격리된 분석 전용 환경에서 작업을 수행합니다.

## 4. 요약 및 결론
난독화 해제는 실행 직전의 평문 코드를 포착하는 것이 관건입니다. IEX 치환과 디버깅 기술은 분석가에게 매우 유용한 도구입니다.
