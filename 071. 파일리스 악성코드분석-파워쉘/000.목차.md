# 071. 파일리스 악성코드분석-파워쉘

## 교육 과정 개요
윈도우 관리 도구인 파워쉘(PowerShell)을 악용한 악성 스크립트의 난독화 기법을 해제하고, 메모리 내에서 수행되는 악성 행위를 분석하는 기술을 습득합니다.

## 상세 목차
1. **파워쉘 공격 프레임워크 개요**
   - PowerShell Empire, PowerSploit 등 주요 공격 도구 분석
   - 파워쉘이 공격자에게 매력적인 이유와 권한 체계
2. **파워쉘 스크립트 난독화 기술 분석**
   - Base64 인코딩, 문자열 치환, Backtick 활용 기법
   - 다중 레이어 난독화(Layered Obfuscation) 구조 이해
3. **난독화 해제(De-obfuscation) 실습**
   - `Invoke-Expression(IEX)` 추적 및 실제 실행 코드 추출
   - 파워쉘 디버깅 및 로깅 기능을 이용한 코드 복원
4. **메모리 인젝션 및 쉘코드 실행 분석**
   - `Add-Type` 및 `Reflective DLL Injection` 기법 분석
   - 파워쉘을 이용한 API 호출(Win32 API) 프로세스 이해
5. **WMI와 파워쉘의 결합 공격**
   - WMI 클래스를 이용한 악성 페이로드 은닉 및 실행
   - 원격 관리 기능(WinRM)을 이용한 내부 확산 분석
6. **파워쉘 로깅 및 모니터링 강화**
   - 스크립트 블록 로깅(Event ID 4104) 활성화 및 분석
   - 실행 정책(Execution Policy) 우회 기법과 방어 전략
7. **AMSI 우회 기술과 탐지 전략**
   - 메모리 내 AMSI 패치 기법 분석
   - 행위 기반 파워쉘 공격 탐지 룰 설계
