# 06. 보안 탄력성 측정 지표 및 거버넌스

## 상세 설명
보안 체계가 얼마나 탄력적으로 작동하고 있는지 객관적인 지표로 측정하고 이를 관리하는 거버넌스 체계를 수립합니다.
- **주요 지표(KPI):**
  - MTTD (Mean Time to Detect): 위협 탐지까지 걸리는 평균 시간
  - MTTR (Mean Time to Recover): 사고 발생 후 복구까지 걸리는 평균 시간
  - MTTI (Mean Time to Investigate): 분석 완료까지 걸리는 평균 시간
- **거버넌스:** 보안 탄력성을 유지하기 위한 지속적인 모니터링, 모델 성능 평가, 정책 업데이트 프로세스를 운영합니다.

## 핵심 요약
- 데이터 기반의 지표 관리는 보안 투자 대비 효과를 입증하는 중요한 수단입니다.
- MTTR 단축은 보안 탄력성의 가장 중요한 목표 중 하나입니다.
- 지속적인 학습과 개선이 이루어지는 피드백 루프를 구축해야 합니다.
