# 05. AI 기반 보안 오케스트레이션 및 대응(SOAR)

## 상세 설명
예측 모델이 탐지한 결과를 바탕으로 실제 대응 조치를 자동화하는 SOAR(Security Orchestration, Automation and Response) 기술을 적용합니다.
- **Playbook 연동:** 예측된 위협 시나리오별로 대응 단계(Step-by-step)를 정의한 플레이북을 작성하고 AI가 상황에 맞춰 최적의 플레이북을 선택하게 합니다.
- **오케스트레이션:** 방화벽, EDR, IAM 등 서로 다른 보안 솔루션들을 하나의 워크플로우로 묶어 협업하게 합니다.
- **자동 복구:** 침해된 리소스를 정상 상태의 스냅샷으로 복구하거나 깨끗한 이미지로 재배포하는 과정을 자동화합니다.

## 핵심 요약
- SOAR는 보안 인력의 수동 작업을 자동화하여 대응 속도를 획기적으로 높입니다.
- 지능형 플레이북은 복잡한 위협 상황에서도 표준화된 대응을 보장합니다.
- AI는 최적의 대응 전략을 제안하거나 직접 실행하는 역할을 수행합니다.
