# 70620.5 동적 분석 및 행위 추적

## 1. 동적 분석의 목적
동적 분석은 악성코드를 실제로 실행하여 정적 분석만으로는 파악하기 어려운 실제 행위(네트워크 연결, 파일 생성, 프로세스 인젝션 등)를 실시간으로 모니터링하는 과정입니다.

## 2. 주요 행위 추적 지표

### 2.1 파일 시스템 변화 및 속성 조작
드롭퍼는 페이로드를 드롭한 후 사용자나 보안 솔루션의 눈에 띄지 않게 하기 위해 파일 속성을 변경합니다.
*   **숨김 속성:** `attrib +h` 명령이나 `SetFileAttributes` API를 사용하여 파일을 숨깁니다.
*   **타임스탬프 조작(Timestomping):** 생성된 파일의 시간을 정상 시스템 파일과 유사하게 조작하여 타임라인 분석을 방해합니다.

### 2.2 프로세스 트리 분석
드롭퍼가 어떤 하위 프로세스를 생성하고, 그 자식 프로세스가 다시 어떤 행위를 하는지 구조적으로 파악해야 합니다.
*   **이상 징후:** Word 문서(WINWORD.EXE)가 PowerShell을 실행하거나, 드롭퍼가 시스템 유틸리티를 인젝션 대상으로 사용하는 경우.

## 3. [Python] 실시간 프로세스 및 파일 모니터링 (개념)
`psutil` 및 `watchdog` 라이브러리를 활용하여 시스템의 변화를 감시하는 간단한 분석 도구 구조입니다.

```python
import psutil
import time
import os

def monitor_processes():
    print("[*] Starting process monitoring...")
    seen_pids = set()
    
    try:
        while True:
            for proc in psutil.process_iter(['pid', 'name', 'ppid', 'exe']):
                if proc.info['pid'] not in seen_pids:
                    # 새로운 프로세스 탐지
                    print(f"[NEW PROCESS] PID: {proc.info['pid']} | Name: {proc.info['name']} | Parent: {proc.info['ppid']}")
                    if proc.info['exe']:
                        print(f"  - Path: {proc.info['exe']}")
                    seen_pids.add(proc.info['pid'])
            time.sleep(1)
    except KeyboardInterrupt:
        print("[*] Monitoring stopped.")

def check_file_attributes(file_path):
    if os.path.exists(file_path):
        stats = os.stat(file_path)
        print(f"[*] Analyzing file: {file_path}")
        print(f"  - Hidden: {bool(stats.st_file_attributes & 2) if os.name == 'nt' else 'N/A'}")
        print(f"  - Created Time: {time.ctime(stats.st_ctime)}")

if __name__ == "__main__":
    # 실제 분석 시에는 별도의 스레드나 프로세스로 실행
    monitor_processes()
```

## 4. 동적 분석 도구 활용 실무
실제 사고 분석 현장에서는 다음과 같은 도구들을 조합하여 사용합니다.

*   **Process Monitor (ProcMon):** 파일 시스템, 레지스트리, 네트워크, 프로세스 활동의 모든 이벤트를 캡처합니다. 필터 기능을 활용하여 특정 프로세스 이름이나 경로로 좁혀 분석합니다.
*   **Process Explorer:** 현재 실행 중인 프로세스의 트리 구조, 핸들, 로드된 DLL 정보를 확인합니다.
*   **Any.Run / Joe Sandbox:** 클라우드 기반 샌드박스로, 자동화된 행위 보고서와 네트워크 패킷(PCAP)을 제공합니다.

### 분석 시 유의사항
*   **안티-샌드박스 우회:** 많은 드롭퍼가 마우스 움직임, 특정 파일 존재 여부, 실행 시간 등을 체크하여 가상 환경인 경우 동작하지 않습니다.
*   **네트워크 고립:** 분석용 악성코드가 실제 외부로 통신하지 않도록 가상 네트워크 환경(FakeNet-NG 등)을 구축해야 합니다.
