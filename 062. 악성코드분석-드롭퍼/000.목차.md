# 062. 악성코드분석-드롭퍼

## 교육 과정 개요
악성 바이너리 내부에 다른 악성코드를 숨겨두었다가 특정 조건에서 실행하는 '드롭퍼(Dropper)'의 내부 구조를 파악하고, 리소스 추출 및 실행 메커니즘 분석 기술을 습득합니다.

## 상세 목차
1. **드롭퍼 악성코드의 특징 및 다운로더와의 차이점**
   - 리소스 포함 방식 및 자가 압축 해제 기술의 이해
   - 다단계 실행(Multi-stage execution) 전략 분석
2. **정적 분석: 내장된 리소스 식별 및 추출**
   - PE Resource 섹션 분석 및 숨겨진 데이터 탐색
   - 바이너리 오버레이(Overlay) 데이터 추출 기법
3. **언팩킹(Unpacking) 및 복호화 메커니즘**
   - 런타임에 수행되는 복호화 루틴 역공학 분석
   - 메모리 내에서 생성되는 페이로드 덤프 및 복구
4. **프로세스 인젝션 및 인라인 코드 실행**
   - Process Hollowing 및 DLL Injection 기법 분석
   - 메모리 직접 실행(Fileless-like behavior) 탐지
5. **동적 분석 및 행위 추적**
   - 임시 파일 생성 경로 및 파일 속성 조작 분석
   - 하위 프로세스 생성 트리 및 권한 상승 행위 모니터링
6. **안티-리버싱(Anti-Reversing) 대응**
   - 코드 난독화(Obfuscation) 해제 전략
   - 정크 코드 및 가짜 제어 흐름 분석 방법
7. **드롭퍼 탐지 규칙 설계 및 실습**
   - 행동 기반(Behavioral) 탐지 룰 생성
   - 정적 패턴 매칭을 위한 전문 도구 활용법
