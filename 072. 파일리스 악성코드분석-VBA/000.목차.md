# 072. 파일리스 악성코드분석-VBA

## 교육 과정 개요
MS 오피스 문서에 포함된 VBA(Visual Basic for Applications) 매크로를 이용한 초기 침투 기법을 분석하고, 악성 페이로드 드롭 및 실행 과정을 추적합니다.

## 상세 목차
1. **오피스 매크로 공격의 재부상**
   - 사회 공학(Social Engineering) 기법과 문서 기반 공격
   - 최신 오피스 보안 정책(MOTW 등)과 우회 트렌드
2. **VBA 매크로 구조 및 정적 분석**
   - OLE 구조와 vbaProject.bin 분석 방법
   - `AutoOpen`, `Document_Open` 등 자동 실행 트리거 분석
3. **VBA 난독화 및 은닉 기술**
   - 문자열 결합, ASCII 변환, 가짜 함수 활용 분석
   - 문서 내 숨겨진 컨트롤(UserForms) 및 텍스트 박스 활용 기법
4. **매크로 기반 2차 페이로드 실행 기법**
   - `Shell`, `CreateObject`를 이용한 외부 프로세스 실행
   - 파워쉘 및 스크립트 인터프리터 연동 분석
5. **VBA 및 Windows API 활용 분석**
   - 매크로 내에서 직접 호출되는 Win32 API 추적
   - 프로세스 인젝션 및 가상환경 탐지 루틴 분석
6. **악성 매크로 분석 도구 활용**
   - Oletools(olevba, oledump)를 이용한 신속한 분석
   - 정적 스캔 우회를 위한 P-code 공격 분석
7. **방어 전략 및 사용자 보안 정책**
   - 디지털 서명 및 신뢰할 수 있는 위치 설정
   - 매크로 실행 제한 및 차단 정책 수립 가이드
