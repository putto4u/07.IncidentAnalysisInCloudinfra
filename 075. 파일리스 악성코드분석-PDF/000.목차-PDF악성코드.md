# 075. 파일리스 악성코드분석-PDF

## 교육 과정 개요
PDF 문서의 복잡한 구조와 자바스크립트(JavaScript) 엔진을 악용한 익스플로잇 기법을 분석하고, 숨겨진 악성 요소를 탐지하는 기술을 습득합니다.

## 상세 목차
1. **PDF 파일 아키텍처와 객체(Object) 모델**
   - Header, Body, Cross-Reference Table, Trailer 구조 이해
   - 스트림(Stream) 인코딩 및 압축(FlateDecode, ASCIIHexDecode) 분석
2. **자바스크립트 기반 PDF 공격 분석**
   - `/JS`, `/JavaScript` 액션을 이용한 자동 실행
   - 힙 스프레이(Heap Spraying) 기법을 이용한 메모리 오염 분석
3. **PDF 익스플로잇 분석 및 쉘코드 추출**
   - 특정 버전의 어도비 리더 취약점(UAF 등) 사례 연구
   - 난독화된 자바스크립트 해제 및 페이로드 복원
4. **악성 요소 은닉 기법 분석**
   - `/OpenAction`, `/AA` (Additional Actions) 트리거 분석
   - 내장된 폰트, 이미지 객체 내 데이터 은닉 기법
5. **멀티미디어 및 외부 연결 공격**
   - `/Launch` 액션을 이용한 외부 실행 파일 구동
   - Flash(SWF) 객체 내장 및 취약점 활용 기법
6. **PDF 분석 전용 도구 활용 실습**
   - `pdf-parser`, `peepdf`, `PDFStreamDump` 활용법
   - 정적 분석 우회를 위한 다중 레이어 인코딩 해제
7. **방어 전략 및 뷰어 보안 설정**
   - 자바스크립트 실행 비활성화 및 보안 모드 적용
   - 문서 샌드박싱 기술과 안전한 PDF 관리 가이드
