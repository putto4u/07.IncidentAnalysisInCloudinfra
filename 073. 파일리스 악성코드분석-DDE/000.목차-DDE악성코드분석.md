# 073. 파일리스 악성코드분석-DDE

## 교육 과정 개요
MS 오피스의 DDE(Dynamic Data Exchange) 기능을 악용하여 매크로 없이 외부 명령을 실행하는 공격 기법을 분석하고 탐지 방안을 학습합니다.

## 상세 목차
1. **DDE(Dynamic Data Exchange) 기술의 이해**
   - 윈도우 애플리케이션 간 데이터 공유 메커니즘
   - 매크로(VBA) 기반 공격과의 차이점 및 장단점
2. **DDE 기반 공격 시나리오 분석**
   - MS 워드 및 엑셀 내 DDE 필드 삽입 기법
   - 사용자 클릭을 유도하는 사회 공학적 유포 방식
3. **악성 DDE 필드 구조 및 명령어 분석**
   - `DDEAUTO` 필드를 이용한 자동 실행 메커니즘
   - 외부 프로세스(cmd.exe, powershell.exe) 호출 문법 분석
4. **DDE 공격의 은닉 및 우회 기법**
   - 필드 코드 분할 및 제어 문자 삽입을 통한 탐지 우회
   - 원격 템플릿 주입(Remote Template Injection)과의 결합
5. **동적 분석을 통한 페이로드 추적**
   - 프로세스 모니터링을 통한 하위 프로세스 생성 트리 분석
   - 네트워크 연결 시도 및 C2 통신 식별
6. **DDE 공격 탐지 및 포렌식**
   - 문서 내 `DDE` 관련 키워드 및 필드 정적 스캐닝
   - 윈도우 이벤트 로그를 통한 공격 흔적 추적
7. **방어 및 대응 가이드**
   - 레지스트리 설정을 통한 DDE 기능 비활성화
   - 오피스 보안 업데이트 및 사용자 알림 설정 최적화
