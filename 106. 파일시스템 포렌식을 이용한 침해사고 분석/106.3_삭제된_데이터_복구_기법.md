# 106.3 삭제된 데이터 복구 기법 (Data Recovery)

## 1. 파일 시스템 메타데이터 기반 복구
메타데이터가 살아있는 경우 가장 완벽한 복구가 가능합니다.

### 1.1 삭제 플래그(Delete Flag) 분석
- 파일이 삭제되면 데이터 자체가 지워지는 것이 아니라, MFT 엔트리나 Inode의 '사용 중' 플래그만 해제됨.
- 해당 영역이 덮어쓰기(Overwrite) 되기 전까지는 원본 이름과 경로를 포함한 복구가 가능함.

## 2. 파일 카빙 (File Carving) 기술
메타데이터가 파괴되거나 파일시스템 형식이 지원되지 않을 때 사용하는 기법입니다.

### 2.1 매직 넘버(Magic Number) 및 시그니처 분석
- 파일의 시작(Header)과 끝(Footer)을 나타내는 고유한 바이트 패턴을 검색.
- 예: JPEG (`FF D8 FF E0`), PDF (`25 50 44 46`).
- **한계**: 파일이 단편화(Fragmentation)되어 저장된 경우 복구율이 급격히 떨어짐.

### 2.2 고급 카빙 기법
- 파일 내부 구조 분석을 통한 유효성 검증 및 단편화된 조각 맞추기 기술.
