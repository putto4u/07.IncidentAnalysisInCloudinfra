# 106.7 파일시스템 포렌식 도구 활용 실습

## 1. 주요 포렌식 툴셋
분석 목적에 맞는 도구 선택과 활용법을 학습합니다.

### 1.1 Sleuth Kit (TSK) & Autopsy
- **TSK**: 커맨드라인 기반의 강력한 파일시스템 분석 도구 모음 (`fls`, `icat`, `ils` 등).
- **Autopsy**: TSK를 기반으로 한 GUI 분석 플랫폼. 타임라인 생성 및 해시 분석 기능 제공.

### 1.2 데이터 추출 및 이미징
- **FTK Imager**: 증거 원본의 무결성을 유지하며 디스크 이미지를 생성하고 내부 구조를 신속하게 확인.
- **HxD (Disk Editor)**: 저수준 섹터 에디팅을 통해 파일 시스템 구조를 직접 확인하고 수동으로 데이터 추출.

## 2. 실습 시나리오
- 포맷된 USB 메모리에서 삭제된 기밀 문서 복구하기.
- NTFS $LogFile 분석을 통해 특정 시간에 삭제된 파일 리스트 추출하기.
- ADS에 숨겨진 악성 코드 찾아내기.
