# 106.6 파일시스템 안티포렌식 탐지

## 1. 데이터 은닉 기법 탐지
공격자가 분석을 방해하기 위해 사용하는 수법들을 파악합니다.

### 1.1 ADS (Alternate Data Streams) 분석
- NTFS의 특성을 이용하여 하나의 파일명에 여러 개의 데이터 스트림을 숨기는 기법.
- `dir /r` 명령어나 전용 분석 도구로 숨겨진 악성 페이로드 탐지.

### 1.2 불량 섹터(Bad Sector) 마킹
- 정상 섹터를 고의로 불량 섹터로 표시하여 포렌식 도구의 접근을 차단하고 데이터를 숨기는 기법.

## 2. 타임스탬프 조작 (Timestomping) 검증
- **일관성 검증**: NTFS의 $SI 타임스탬프와 $FN 타임스탬프를 비교. 일반적으로 조작 도구는 $SI만 변경하므로 두 정보의 차이를 통해 조작 여부 판별.
- **MFT 수정 시간 분석**: 파일 내용은 수정되지 않았는데 MFT 엔트리 수정 시간만 변경된 경우 등 비정상 패턴 분석.
