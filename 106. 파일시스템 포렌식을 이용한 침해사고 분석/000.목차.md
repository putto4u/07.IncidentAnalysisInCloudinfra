# 106. 파일시스템 포렌식을 이용한 침해사고 분석

## 교육 과정 개요
데이터가 저장되는 근본적인 구조인 파일시스템의 저수준(Low-level) 분석을 통해 삭제된 흔적을 복구하고 숨겨진 데이터를 찾아내는 심층 분석 기술을 학습합니다.

## 상세 목차
1. **파일시스템 구조와 데이터 저장 원리**
   - 섹터(Sector), 클러스터(Cluster), 슬랙 공간(Slack Space)의 이해
   - 주요 파일시스템(NTFS, FAT32, Ext4, APFS) 구조 비교
2. **NTFS 파일시스템 심층 분석**
   - MFT(Master File Table) 구조 및 속성($MFT, $LOGFILE, $USNjrnl) 분석
   - 상주 데이터(Resident Data) 및 비상주 데이터 처리 메커니즘
3. **삭제된 데이터 복구 기법 (Data Recovery)**
   - 파일 카빙(File Carving) 기술과 매직 넘버(Magic Number) 활용
   - 메타데이터 분석을 통한 원본 파일명 및 경로 복원
4. **저널링(Journaling) 시스템 분석**
   - 파일 시스템 변경 이력 추적을 통한 공격 행위 시간대 특정
   - 트랜잭션 로그 분석을 통한 삭제된 흔적 재구성
5. **파티션 및 볼륨 분석**
   - MBR/GPT 파티션 구조와 숨겨진 영역(HPA, DCO) 탐색
   - 가상 디스크(VHD, VMDK) 및 RAID 구성 환경의 포렌식 접근
6. **파일시스템 안티포렌식 탐지**
   - 불량 섹터(Bad Sector) 마킹을 이용한 데이터 은닉 탐지
   - 타임스탬프 조작 여부 확인을 위한 일관성 검증
7. **파일시스템 포렌식 도구 활용 실습**
   - Autopsy, FTK Imager, Sleuth Kit(TSK) 활용법
   - 저수준 디스크 에디터를 이용한 수동 데이터 추출
