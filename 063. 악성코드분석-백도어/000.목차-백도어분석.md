# 063. 악성코드분석-백도어

## 교육 과정 개요
침해된 시스템에 지속적으로 접근하기 위해 설치되는 '백도어(Backdoor)' 및 원격 제어 도구(RAT)의 통신 방식과 은닉 기술을 심층 분석하고 추적 기술을 학습합니다.

## 상세 목차
1. **백도어 및 RAT(Remote Access Trojan)의 진화**
   - 시스템 제어권 획득 및 원격 관리 기능의 이해
   - 상용 도구 기반 백도어와 커스텀 백도어의 차이
2. **지속성 유지 기법 (Persistence Mechanics)**
   - 서비스 등록, 작업 스케줄러, 레지스트리 자동 실행 분석
   - COM 핸들러 및 WMI 이벤트 구독을 이용한 은닉
3. **C2(Command & Control) 통신 프로토콜 심층 분석**
   - 비표준 포트 활용 및 커스텀 암호화 프로토콜 분석
   - DNS 터널링, HTTP 파이프라인 등 우회 통신 기법
4. **원격 명령 실행 및 정보 탈취 기능 분석**
   - 쉘 명령어 실행, 파일 전송, 화면 캡처 모듈 분석
   - 키로깅 및 오디오/비디오 스트리밍 기능 식별
5. **권한 상승(Privilege Escalation) 분석**
   - 로컬 취약점(UAC Bypass 등) 활용 기법 역공학
   - 시스템 계정(SYSTEM/Admin) 획득 프로세스 추적
6. **탐지 우회 및 루트킷 기능 분석**
   - 파일/프로세스/포트 은닉을 위한 API 후킹 기술
   - 커널 모드 백도어의 기본 동작 원리 이해
7. **포렌식 관점의 백도어 추적 및 제거**
   - 메모리 포렌식을 통한 활성 백도어 식별
   - 타임라인 분석을 이용한 백도어 설치 시점 특정
