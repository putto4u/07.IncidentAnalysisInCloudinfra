# 103.7 실전 네트워크 침해사고 시나리오 분석

## 1. 전형적인 공격 시나리오: APT 공격
1. **Initial Access:** 피싱 메일을 통한 악성 코드 다운로드 (HTTP/HTTPS 로그 분석).
2. **C2 Connection:** 외부 서버와 지속적 연결 확립 (비컨 분석).
3. **Internal Reconnaissance:** 내부망 스캔 및 취약한 자산 탐색 (ARP, ICMP, 포트 스캔 트래픽 분석).
4. **Lateral Movement:** 관리자 계정 탈취 후 다른 서버로 전파 (SMB, RDP, SSH 로그 분석).
5. **Data Exfiltration:** 중요 DB 정보 수집 후 외부 전송 (대량 세션 분석).

## 2. 시나리오별 핵심 분석 포인트
### 2.1 랜섬웨어 유입
- 특정 IP에서 다수의 파일 확장자가 변경되는 트래픽이 발생하는지 확인.
- 랜섬웨어 유포지로 알려진 도메인 접속 이력 추적.

### 2.2 내부자에 의한 유출
- 인가된 사용자가 평소에 접근하지 않던 서버에 접근하여 대량의 데이터를 가져가는 패턴 식별.

## 3. 사후 대응 및 교훈 (Lessons Learned)
- 분석 결과를 바탕으로 방화벽 규칙을 강화하고, 유사한 패턴을 탐지할 수 있는 IDS/IPS 룰(Snort, Suricata)을 생성하여 적용합니다.
