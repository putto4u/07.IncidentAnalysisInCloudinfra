# 103. 네트워크포렌식을 이용한 침해사고분석

## 교육 과정 개요
네트워크 트래픽 데이터(Packet, Flow, Log)를 분석하여 공격자의 유입 경로, 통신 패턴, 데이터 유출 흔적을 추적하는 네트워크 포렌식 기술을 학습합니다.

## 상세 목차
1. **네트워크 포렌식 아키텍처 및 데이터 소스**
   - 실시간 패킷 캡처(Full Packet Capture)와 넷플로우(NetFlow) 비교
   - IDS/IPS, 방화벽, 웹 프록시 로그의 연계 분석 가치
2. **트래픽 수집 및 패킷 분석 기술**
   - Wireshark 및 Tshark를 이용한 대용량 PCAP 분석 기법
   - 계층별 프로토콜(TCP/IP, HTTP, DNS, SMB) 이상 행위 식별
3. **C2 통신 및 비정상 트래픽 추적**
   - 비표준 포트 및 암호화 터널링(VPN, SSH) 분석
   - 비컨(Beacon) 패턴 매칭을 통한 좀비 PC 식별
4. **네트워크 기반 데이터 유출(Exfiltration) 탐지**
   - DNS 쿼리, ICMP 페이로드 등을 이용한 데이터 은닉 전송 분석
   - 대용량 데이터 전송 세션 식별 및 목적지 IP 추적
5. **암호화 트래픽 분석 전략**
   - SSL/TLS 핸드쉐이크 분석 및 인증서 검증 여부 확인
   - 복호화 키 확보를 통한 HTTPS 페이로드 가시성 확보
6. **네트워크 타임라인 분석 및 시각화**
   - 공격 단계별 네트워크 이벤트 발생 순서 재구성
   - 그래프 데이터베이스를 활용한 공격 경로 시각화
7. **실전 네트워크 침해사고 시나리오 분석**
   - 내부망 침투 후 Lateral Movement 과정의 네트워크 흔적 분석
