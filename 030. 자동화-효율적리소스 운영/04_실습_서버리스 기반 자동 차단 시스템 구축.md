# 4.1 실습: 서버리스 기반 자동 차단 시스템 구축

## 상세 설명
본 실습에서는 클라우드 로그에서 비정상 활동을 감지하고, 해당 공격 IP를 방화벽에서 자동으로 차단하는 시스템을 구축합니다.
- **시나리오:** 특정 IP에서 짧은 시간 내에 다량의 403 에러(권한 없음)가 발생할 경우 이를 무차별 대입 공격 또는 스캔으로 간주합니다.
- **구성 요소:**
  1. CloudWatch Logs / Cloud Logging: 로그 수집 및 필터링
  2. Lambda / Cloud Functions: 차단 로직 수행 (Python Boto3 등 활용)
  3. WAF / Cloud Armor: IP Set 업데이트 및 차단 적용
- **핵심 코드:** 최신 SDK(Python 3.12+)를 사용하여 차단 리스트를 동적으로 관리하는 스크립트를 작성합니다.

## 핵심 요약
- 로그 분석부터 차단까지의 전 과정을 서버리스 아키텍처로 구현합니다.
- Boto3 등 클라우드 SDK의 최신 API 활용 능력을 배양합니다.
- 실시간 위협에 대응하는 방어 자동화 역량을 실습합니다.
