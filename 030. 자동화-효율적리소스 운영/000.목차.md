### **📖 핵심 내용**

#### **1. eBPF 기반 런타임 보안 (Falco 활용)**
[cite_start]2026년 현재, 컨테이너 보안의 표준은 커널 레벨에서의 가시성 확보입니다[cite: 21]. [cite_start]**Falco**를 통해 컨테이너 내부의 이상 징후를 탐지합니다[cite: 22].


#### **2. Python 기반 자동 대응 스크립트 (Boto3 & Google Cloud SDK 2026)**
[cite_start]보안 경보 발생 시, 특정 IAM 사용자의 권한을 즉시 무효화하거나 공격 IP를 클라우드 방화벽(WAF/Cloud Armor)에서 차단하는 자동화 코드입니다[cite: 23].
* [cite_start]**기술 스택:** Python 3.12+, Boto3 v1.34+, google-cloud-iam v2.15+[cite: 23].
* [cite_start]**특이사항:** `Deprecated`된 정책 직접 수정 방식 대신, `Quarantine` 프로젝트/그룹 이동 또는 `Session Invalidation`을 권장합니다[cite: 23].

### **🛠️ 실습 문제: 자동화된 IP 차단 워크플로우**

[cite_start]**[문제]** CloudWatch Logs(AWS) 또는 Cloud Logging(GCP)에서 특정 IP가 1분 내에 100회 이상의 403 에러를 발생시켰을 때, 해당 IP를 클라우드 방화벽 ACL에 자동으로 등록하여 1시간 동안 차단하는 서버리스 함수를 완성하시오[cite: 24].

**[조건]**
1.  [cite_start]**Boto3** 또는 **Google Cloud Client Library**의 최신 API를 사용할 것[cite: 25].
2.  [cite_start]차단 리스트는 **WAFv2 API** 또는 **Cloud Armor API**를 이용해 IP Set을 업데이트할 것[cite: 25].
3.  [cite_start]**GitHub Reference:** [AWS SDK for Python (Boto3)](https://github.com/boto/boto3) / [Google Cloud Python Client](https://github.com/googleapis/google-cloud-python)[cite: 26].
