# 030. 자동화-효율적리소스 운영 - 04. [실습] 서버리스 기반 자동 차단 시스템 구축

## 개요
이 실습에서는 서버리스 아키텍처를 활용하여 클라우드 환경에서 발생하는 비정상적인 IP 활동을 자동으로 탐지하고 차단하는 시스템을 구축합니다. AWS Lambda (또는 GCP Cloud Functions)와 클라우드 방화벽(WAFv2 또는 Cloud Armor)을 연동하여 실시간 자동 대응 능력을 강화합니다.

## 4.1 실습 환경: AWS/GCP 환경에서의 비정상 IAM 활동 탐지 시나리오
실습은 AWS 또는 GCP 환경에서 진행되며, 특정 IP 주소에서 발생하는 비정상적인 IAM 활동(예: 짧은 시간 내에 다수의 인증 실패 또는 권한 없는 접근 시도)을 탐지하는 시나리오를 가정합니다.
- **클라우드 플랫폼:** AWS 또는 GCP (선택)
- **탐지 대상:** 특정 IP 주소에서 발생하는 403 Forbidden 에러 (인증/권한 실패)
- **모니터링 서비스:**
    - AWS: CloudWatch Logs, CloudTrail
    - GCP: Cloud Logging

## 4.2 실습 과제: 파이썬 최신 라이브러리를 활용한 IP 차단 및 세션 강제 종료 스크립트 작성
주어진 시나리오에 따라, 탐지된 비정상 IP를 클라우드 방화벽에서 자동으로 차단하고, 필요한 경우 비정상적인 세션을 강제 종료하는 파이썬 스크립트를 작성합니다.

### 실습 문제: 자동화된 IP 차단 워크플로우
CloudWatch Logs(AWS) 또는 Cloud Logging(GCP)에서 특정 IP가 1분 내에 100회 이상의 403 에러를 발생시켰을 때, 해당 IP를 클라우드 방화벽 ACL에 자동으로 등록하여 1시간 동안 차단하는 서버리스 함수를 완성하시오.

### 조건
1.  **Boto3** (AWS) 또는 **Google Cloud Client Library** (GCP)의 최신 API를 사용할 것.
2.  차단 리스트는 **WAFv2 API** (AWS) 또는 **Cloud Armor API** (GCP)를 이용해 IP Set을 업데이트할 것.
3.  **GitHub Reference:**
    -   [AWS SDK for Python (Boto3)](https://github.com/boto/boto3)
    -   [Google Cloud Python Client](https://github.com/googleapis/google-cloud-python)

### 실습 가이드 (AWS 예시)

#### 1. CloudWatch Logs 필터 설정
- CloudTrail에서 403 에러를 발생시키는 이벤트를 CloudWatch Logs로 전송하도록 설정합니다.
- CloudWatch Logs에서 특정 IP가 1분 내에 100회 이상의 403 에러를 발생시키는 패턴을 필터링하는 Metric Filter를 생성합니다.

#### 2. AWS Lambda 함수 개발 (Python)
- **함수 트리거:** CloudWatch Logs Metric Filter에서 생성된 경보(Alarm)를 트리거로 설정합니다.
- **Boto3 사용:**
    - `boto3` 라이브러리를 사용하여 WAFv2 API를 호출합니다.
    - `get_ip_set`, `update_ip_set` 등의 API를 활용하여 IP Set에 IP 주소를 추가합니다.
    - IP 차단 후 1시간 뒤에 IP를 자동으로 제거하는 로직을 구현합니다 (예: DynamoDB 또는 S3에 만료 시간과 함께 저장 후 EventBridge Scheduler로 주기적으로 확인).

```python
import json
import boto3
import os
from datetime import datetime, timedelta

def lambda_handler(event, context):
    print(f"Received event: {json.dumps(event)}")

    # 예시: CloudWatch Alarms에서 넘어온 IP 추출 로직 (실제 구현에 따라 달라질 수 있음)
    # 실제 환경에서는 CloudWatch Alarm의 Payload에서 필요한 정보를 파싱해야 합니다.
    # 여기서는 예시를 위해 가상의 IP 주소를 사용합니다.
    threat_ip = "192.0.2.1" # 실제 이벤트에서 추출해야 합니다.

    region = os.environ.get('AWS_REGION', 'ap-northeast-2')
    waf_client = boto3.client('wafv2', region_name=region)

    ip_set_name = os.environ.get('IP_SET_NAME', 'ThreatIPSet')
    scope = os.environ.get('WAF_SCOPE', 'REGIONAL') # CLOUDFRONT 또는 REGIONAL

    try:
        # 1. IP Set ID와 Lock Token 가져오기
        list_ip_sets_response = waf_client.list_ip_sets(Scope=scope)
        ip_set_id = None
        lock_token = None

        for ip_set in list_ip_sets_response['IPSets']:
            if ip_set['Name'] == ip_set_name:
                ip_set_id = ip_set['Id']
                # IP Set 상세 정보를 가져와 LockToken 추출
                get_ip_set_response = waf_client.get_ip_set(Name=ip_set_name, Scope=scope, Id=ip_set_id)
                lock_token = get_ip_set_response['LockToken']
                break

        if not ip_set_id or not lock_token:
            print(f"IP Set '{ip_set_name}' not found or LockToken could not be retrieved.")
            return {
                'statusCode': 500,
                'body': json.dumps('IP Set not found or LockToken error')
            }

        # 2. IP Set에 IP 주소 추가
        # 이미 존재하는 IP인지 확인 후 추가 (생략 가능, WAF가 중복 처리)
        # 현재 IP Set 내용을 가져와서 threat_ip가 없는 경우에만 추가하는 로직이 더 안전
        update_ip_set_response = waf_client.update_ip_set(
            Name=ip_set_name,
            Scope=scope,
            Id=ip_set_id,
            Addresses=[f"{threat_ip}/32"], # CIDR 형식으로 추가
            LockToken=lock_token
        )
        print(f"Successfully added {threat_ip} to IP Set {ip_set_name}. Response: {update_ip_set_response}")

        # 3. (선택) DynamoDB에 차단된 IP와 만료 시간 기록하여 1시간 후 자동 제거 로직 구현
        # 이 부분은 추가적인 서비스(EventBridge Scheduler, DynamoDB TTL)와 연동하여 구현
        # 예시:
        # dynamodb = boto3.resource('dynamodb', region_name=region)
        # table = dynamodb.Table('BlockedIPs')
        # expiration_time = int((datetime.now() + timedelta(hours=1)).timestamp())
        # table.put_item(
        #     Item={
        #         'IPAddress': threat_ip,
        #         'ExpiresAt': expiration_time
        #     }
        # )

    except Exception as e:
        print(f"Error blocking IP {threat_ip}: {e}")
        return {
            'statusCode': 500,
            'body': json.dumps(f"Error blocking IP: {str(e)}")
        }

    return {
        'statusCode': 200,
        'body': json.dumps('IP blocking workflow completed.')
    }

```
