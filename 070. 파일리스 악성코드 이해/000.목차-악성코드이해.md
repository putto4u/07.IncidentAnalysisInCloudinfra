# 070. 파일리스 악성코드 이해

## 교육 과정 개요
전통적인 실행 파일 형식을 벗어나 시스템의 정상 도구와 메모리를 활용하는 파일리스(Fileless) 공격의 개념을 정립하고 전체적인 공격 프레임워크를 이해합니다.

## 상세 목차
1. **파일리스 공격의 정의와 필요성**
   - 시그니처 기반 보안 솔루션의 한계와 공격자의 선택
   - 'Living off the Land(LotL)' 기법의 개념 이해
2. **파일리스 공격의 주요 유형 분석**
   - 메모리 상주형(Memory-only) 악성코드
   - 스크립트 기반(PowerShell, WMI, VBS) 공격
   - 레지스트리 상주형 공격 기법
3. **공격 매커니즘: 초기 침투에서 실행까지**
   - 매크로 포함 문서 파일을 통한 초기 진입
   - 익스플로잇 킷을 이용한 브라우저 메모리 오염
4. **정상 시스템 도구(LOLBins) 활용 사례**
   - PowerShell, Wmic, Mshta, Certutil 등 주요 도구의 악용 방식
   - 합법적인 도구를 이용한 권한 상승 및 정보 탈취
5. **파일리스 공격의 지속성 확보 전략**
   - WMI 이벤트 구독(Event Subscription) 활용
   - 레지스트리 Run 키에 인코딩된 스크립트 삽입
6. **탐지 및 분석의 어려움과 대응 방안**
   - 파일 검사 중심 보안의 맹점 분석
   - AMSI(Antimalware Scan Interface)의 역할과 한계
7. **차세대 보안 솔루션(EDR/XDR)의 가시성 확보**
   - 행위 로그 기반의 상관관계 분석 중요성
   - 비정상적인 부모-자식 프로세스 관계 추적
