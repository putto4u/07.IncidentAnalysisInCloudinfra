# 정상 시스템 도구(LOLBins) 활용 사례
## 1. 개요
LOLBins(Living off the Land Binaries)는 공격자가 악성 행위를 위해 사용하는 운영체제 내장 정상 실행 파일들을 의미합니다.

## 2. 상세 분석 내용
### PowerShell, Wmic, Mshta, Certutil 등 주요 도구의 악용 방식
- **Certutil:** 원래 인증서 관리 도구이나, 원격 파일을 다운로드하거나 인코딩된 데이터를 디코딩하는 데 악용됩니다.
- **Mshta:** HTA(HTML Application) 파일을 실행하는 도구로, 원격의 악성 스크립트를 실행하는 통로로 쓰입니다.
### 합법적인 도구를 이용한 권한 상승 및 정보 탈취
- **권한 상승:** 정상 도구의 취약점이나 설정 오류를 이용하여 관리자 권한을 획득합니다.
- **우회 실행:** 보안 솔루션이 차단하는 실행 파일 대신, 허용된 정상 도구를 통해 동일한 기능을 수행합니다.

## 3. 대응 및 방어 전략
- **LOLBins 모니터링:** 평소 업무에 쓰이지 않는 정상 도구의 실행이나, 비정상적인 인자값(예: certutil의 -urlcache 옵션 등) 사용을 감시합니다.
- **애플리케이션 화이트리스트:** 필요한 도구만 실행 가능하도록 제한하는 정책을 도입합니다.

## 4. 요약 및 결론
정상 도구의 악용은 '양날의 검'과 같습니다. 편리한 도구가 공격의 수단이 되지 않도록 실행 컨텍스트에 대한 세밀한 감시가 필요합니다.
