# 차세대 보안 솔루션(EDR/XDR)의 가시성 확보
## 1. 개요
파일리스 공격에 대응하기 위해 등장한 EDR 및 XDR 솔루션이 어떻게 공격의 가시성을 확보하고 대응하는지 학습합니다.

## 2. 상세 분석 내용
### 행위 로그 기반의 상관관계 분석 중요성
- **이벤트 체인:** 단순히 '파워쉘 실행'이 아닌, '워드 문서 -> 파워쉘 실행 -> 외부 IP 접속 -> 파일 암호화'로 이어지는 전체 행위 체인을 분석합니다.
- **비정상 패턴 탐지:** 평소와 다른 정상 도구의 동작이나, 대량의 데이터 유출 시도를 탐지합니다.
### 비정상적인 부모-자식 프로세스 관계 추적
- **프로세스 트리:** 오피스 프로그램이 쉘이나 시스템 도구를 실행하는 것과 같이 일반적이지 않은 부모-자식 관계를 즉각 식별합니다.
- **실시간 격리:** 위협이 감지된 프로세스를 즉시 종료하고 관련 네트워크를 차단하는 대응 기능을 수행합니다.

## 3. 대응 및 방어 전략
- **EDR 도입 및 운영:** 모든 엔드포인트의 행위 데이터를 중앙집중화하여 가시성을 확보합니다.
- **위협 헌팅(Threat Hunting):** 솔루션의 자동 탐지 외에도, 전문가가 로그를 분석하여 숨어있는 위협을 능동적으로 찾아냅니다.

## 4. 요약 및 결론
EDR/XDR은 파일리스 공격에 대한 '디지털 블랙박스' 역할을 합니다. 행위 데이터의 수집과 연계 분석이 현대 보안의 핵심입니다.
